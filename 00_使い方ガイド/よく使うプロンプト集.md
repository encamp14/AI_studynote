## メール系

### メール下書きをトーン調整
```
以下のメール下書きを、次の関係性と目的に合わせて書き直してください。
# 相手との関係: [上司/同僚/取引先/初対面]
# 目的: [報告/依頼/お礼/お断り]
# トーン: [丁寧/簡潔/温かい]

# 下書き
[ここに貼り付け]
```

### 商談後フォローアップメール
```
以下の商談情報をもとに、フォローアップメールを作成してください。
# 商談情報
・相手: [○○様]
・商談日時: [○月○日]
・主な議題: [箇条書き]
・相手の関心ポイント: [○○]
・宿題事項: [自分が持ち帰った課題]
# 求める要素
・お時間を頂いた御礼
・議論の要点を1-2行で再確認
・宿題への対応スケジュール
・次のアクション(資料送付/再面談 など)の提案
・返信を促す結び
```

### 社内向け依頼メール
```
以下の依頼を、社内向けメールにしてください。
# 依頼内容
・宛先: [部署名・役職]
・依頼すること: [具体的に]
・期限: [○月○日まで]
・依頼の背景: [なぜ必要か]
・代替案: [もし難しければ ○○ で OK]
# トーン: 丁寧かつ簡潔、相手の業務状況を配慮した一言を冒頭に
```

### 取引先への断りメール
```
以下の条件で、取引先への断りメールを作成してください。
# 条件
・相手: [株式会社○○の○○様]
・件名: [何のお断り(見積/提案/依頼)]
・理由: [予算/タイミング/方針 など]
・トーン: 非常に丁重に、今後の関係を維持できるように
・必ず入れる要素: 感謝の言葉 / 断りの明確化 / 将来の含み / クッション言葉
・文字数: 300字程度
```

### 外国語メールの翻訳＋返信
```
このメールについて以下をお願いします。
# 1. 日本語訳(意訳でOK、文化的背景の補足あり)
# 2. 主要ポイント3つ
# 3. 私の代わりに返信文を{対象言語}で作成
# 返信内容: [伝えたいこと]
# 文化的配慮: [相手の国・地域を考慮]
```
