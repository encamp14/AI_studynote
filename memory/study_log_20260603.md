### Git/GitHub 深掘り理解（疑問→解決）

| 疑問 | 答え |
|---|---|
| Gitを使うタイミング | ファイルを変更・共有・取り込むとき |
| Gitが得意なもの | テキスト・コード・MD（画像・動画は苦手） |
| GitはClaudeの会話を覚えるか | 覚えない。メモリファイルをGitで管理するのが正解 |
| リポジトリは用途で分けるか | YES。用途ごとに分けるのが正しい |
| GitHubとAI連携 | ClaudeはファイルをGitHubで管理、Cursorは直接連携 |
| コンテキストウィンドウ対策 | 会話内容を.mdに保存→GitHubにpush→新チャットで読み込む |

**コンテキストウィンドウの仕組みまとめ：**
```
会話の内容 → .mdファイルに保存 → GitHubにpush
→ 新しいチャットでそのファイルを読み込ませる
→ Claudeがまた覚えてくれる
```
