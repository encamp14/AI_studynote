# プロンプトテンプレート一覧

---

## 1. マニュアル作成

```
# 業務内容: [○○]
# 対象読者: [新人/異動者/外注先]
# 求める構成
・業務の目的と全体像
・前提条件・必要なツール/権限
・手順(ステップごとに番号付き)
・各ステップの注意点
・トラブル時の対応
・参考資料・関連リンク
# トーン: 平易で具体的に
```

---

## 2. 日報作成

```
今日の日報を作って。

・午前：[やったこと]
・午後：[やったこと]
・気づき：[学んだこと・発見]
・明日：[次にやること]
```

---

## 3. QUERY関数を教えてもらう

```
スプレッドシートでQUERY関数を使いたい。
[○○]を抽出したい。
列の構成は：A列=名前、B列=日付、C列=人数…（自分のシートに合わせて）
QUERY関数を教えて。
```

---

## 4. Gemini Gem（専用プロンプト）を作る

```
以下の条件でGemini Gemのプロンプトを作って。

# 用途: [○○]
# 毎回入力する情報: [日付・人数・プランなど]
# 出力してほしい形式: [帳票/表/文章など]
# 注意点: [○○]
```

---

## 5. クライアント情報をMD形式でまとめる

```
以下の情報をMD形式でまとめて。
他のメンバーがAIに読み込ませてすぐ把握できるようにして。

[会議メモ・経歴・考え方などをここに貼る]
```
